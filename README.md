# Stack Exchange Data Analysis

## Overview
This project analyzes CrossValidated Stack Exchange data using SQL and R.
The analysis investigates user activity, question trends, response behavior,
and statistical topic popularity.

## Tools Used
- R
- SQL
- SQLite
- ggplot2
- DBI
- RSQLite

## Methods
- Relational database queries
- Exploratory data analysis
- Correlation analysis
- Statistical summaries
- Data visualization

## Findings
- Machine learning and regression were among the most common discussion topics.
- A large proportion of questions received few responses.
- User activity and posting behavior changed significantly over time.
